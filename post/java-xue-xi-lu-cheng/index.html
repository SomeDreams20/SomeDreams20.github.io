常用的DOS命令

1.1打开DOS命令窗口

win+R 打开运行窗口，在文本框内输入cmd

1.2查看目录文件 dir命令

输入cmd之后再页面输入dir 回车后，显示图中所显示的当前目录下的所有文件夹。

创建目录命令 mkdir xxx

1.3 切换路径命令 cd

cd即change directory 使用方法是:cd+目录路径，其中路径包括相对路径和绝对路径。

相对路径：相对路径都是从当前所在目录作为起点开始查找的路径，是绝对不会以盘符开始的。

绝对路径：在windows操作系统中凡是起点是磁盘符开始的路径。

例如：C:\Users\Administrator

​ D:\BaiduNetdiskDownload

区别：相对路径是一定是当前位置作为起点开始寻找。

如上图所示，先用mkdir 建立一个 lau的文件，再使用cd lau完成目录的切换。

cd . 表示当前目录

cd .. 表示返回上一级目录

cd \ 表示返回根目录

1.4切换盘符

输入c:/d:/f: 加上空格，然后按回车即可。不需要cd

1.5删除命令del

del命令，删除一个或多个文件

del abc 删除abc文件

del *abc 删除所有以abc结尾的文件

1.6ipconfig命令

ipconfig命令查看当前IP地址等信息，需要查看更详细的信息试用ipconfig/all。

1.7ping命令

A计算机在网络中想要定位到（或连接到）B计算机，首先必须先要知道b的计算机ip地址。

ping ip地址/域名

		ping www.baidu.com
		ping 61.135.169.121

1.8其它一些命令和按键组合

cls 表示清屏

exit 表示退出DOS命令窗口

tab 键表示补全当前文件夹的名字

当访问某一磁盘目下的具体文件输入首字母加上tab即可补全

ctrl + c 复制

ctrl + v 粘贴

ctrl + x 剪切

ctrl + z 撤销

ctrl + y 重做

回到行首home ，回到行尾 end

光标在行尾如何选中整行 shift + home

光标在行首如何选中整行 shift + end

全选 ctrl + A

查询 ctrl + F
